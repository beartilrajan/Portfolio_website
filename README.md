# Beartil Rajan - Developer Portfolio

A modern, high-performance developer portfolio built with React, WebGL fluid interaction dynamics, and vanilla CSS.

## Features

- **Interactive WebGL Fluid Canvas**: Fluid simulation layer confined to the Hero section with dynamic cursor physics and smooth ambient gradient masks.
- **Glassmorphic Aesthetics**: Modern dark mode palette with custom tokens, blurred glass panels, and smooth scroll reveals.
- **Full Responsiveness**: Customized for all device viewports, including mobile devices, tablets, and desktops.
- **Architecture Specs Modal**: Interactive project showcases with deep-dive technical specs.

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Vanilla CSS (Design Tokens & CSS Variables)
- **Visual Effects**: WebGL Fluid Simulation, Lucide Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
